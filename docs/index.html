Weather App

Search Weather Site

This weather app is a project I made for a class I took at Code Louisville. The class was 'Javscript with React.js' and this React App I created starting with create-react-app. In this app I wanted to dispay appealing visuals and simplistic design. Additionally, I wanted to use pastel tones that were pleasing and animations that made the app more interesting than a static page. I still have a few things to add as far as functionality goes with this project, however, I think it gives a lot to my portfolio and was worth adding.
